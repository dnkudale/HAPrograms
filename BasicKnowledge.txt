GIT Process for Setting Repo from Remote (e.g. Github, bitbucket, etc)
	Step 1) git remote -v
	Step 2) git remote add origin (SSH or HTTPS Link., e.g. git@github.com:dnkudale/HAPrograms.git)
	Step 3) git status
	Step 4) git pull
	Step 5) ls
	Step 6) git add .
	Step 7) git commit -m "Message"
	Step 8) git push --set-upstream origin master
